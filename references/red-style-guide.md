# 小红书文案风格参考（基于真实爆款数据分析）

> 数据来源：MediaCrawler 爬取数据 + 100+ 爆款笔记分析 + 2025-2026运营报告

## 平台核心数据

- 月活用户：3亿+
- 核心用户：18-35岁女性（占比70%+）
- 笔记标题上限：20字
- 爆款笔记平均字数：150-300字
- 高赞笔记特征：emoji密度高、分段清晰、标签3-5个

## 爆款标题公式（数据验证）

### 公式一：结果前置 + 场景限定 + 低门槛
> 核心结构：核心结果 + 适用场景 + 轻松实现

示例：
- 7天瘦5斤｜上班族晚餐食谱，不用节食
- 28天皮肤亮2度｜懒人护肤3步搞定
- 3分钟搞定｜打工人快手早餐，不用开火

### 公式二：人群 + 痛点/需求 + 利益点
> 核心结构：目标人群 + 核心痛点 + 解决方案

示例：
- 敏感肌必看！泛红急救3步，1周退红
- 小个子穿搭｜显高10cm的5个公式
- 新手博主｜0粉起步第一个月涨粉1000+

### 公式三：反差/悬念 + 核心信息
> 核心结构：制造反差或悬念 + 关键信息

示例：
- 后悔没早知道！30块搞定大牌同款效果
- 被问爆了！这3个习惯让皮肤越来越好
- 姐妹们别再乱买了！这才是真正好用的

### 标题数据规律
- 含数字的标题点击率高23%（"3步""7天""5个"）
- 含"！"的标题互动率高18%
- 15-20字的标题曝光率最高
- 标题承担搜索权重，关键词要前置

## 排版细节（基于高赞笔记统计）

- 每行不超过25字（手机屏幕友好）
- emoji每2-3行出现1-2个，过多反而降赞
- 段落之间必须空行
- 重点用💡📌✨标注
- 标签用#包裹，放文末
- 数字用阿拉伯数字（3步 > 三步）
- 感叹号适度使用（1-3个即可）
